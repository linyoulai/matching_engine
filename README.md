# Matching Engine

A highly extensible C++ matching engine project supporting multiple order types, modular design, and easy for future expansion and testing.

## Directory Structure
- include/    Header files
- src/        Source code
- tests/      Test code
- build/      Build outputs
- scripts/    Build and utility scripts

## Main Modules
- Order (Order and types)
- OrderBook (Order book)
- Matcher (Matching engine)
- RiskControl (Risk management)
- Logger (Logging)

## Build
Recommended: Use CMake for cross-platform builds.
